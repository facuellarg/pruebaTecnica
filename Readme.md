# Prueba tenica IPCOM

![Logo Golang][gopher]

En este proyecto se encontraran las dos partes de la prueba tecnica requerida por IPCOM.  
En ambos casos se uso la version 1.15 de go.  
En el archivo go.mod se especifican las dependencias extras necesarias, sin embargo estas se descargaran de forma automatica cuando se corra el proyecto. Dentro de cada carpeta se encontrara un readme.md con las indicaciones de como correr dicho programa.
